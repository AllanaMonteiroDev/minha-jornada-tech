# Minha Jornada Tech

Projeto desenvolvido para controle de candidaturas a vagas de emprego.

## Funcionalidades
- Cadastro de vagas
- Armazenamento no navegador (localStorage)
- Exclusão de vagas
- Resumo automático (total, entrevistas, recusadas)
- Filtro por status

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo
Praticar manipulação de DOM, lógica de programação e persistência de dados.
